# email

在阿里云轻量级VPS上运行的脚本，定时发送邮件，内容是关于[贵族学院](https://asina.vip/)的。
有获取最新消息和保存缓存等基本功能。

 |              选项 |      参数       | 功能               |
 |: ---------------: | :-------------: | :----------------- |
 |        -h, --help |                 | 帮助
 |        -s, --save | 'y', 'Y', 'yes' | 保存网页内容到缓存 |
 | -f, --flush-cache | 'y', 'Y', 'yes' | 获取最新网页内容   |
